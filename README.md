# 🏏 IPL Data Analysis Dashboard (2008–2024)

An end-to-end data analytics project exploring 17 seasons of the Indian Premier League (IPL) using Python and Power BI. This project analyzes player performances, team success, toss decisions, and match outcomes across 1,095 IPL matches.

## 📊 Dashboard Preview

![Dashboard](dashboard.png)

## 🔍 Key Insights

* 🏆 Mumbai Indians are the most successful IPL franchise with **144 match wins**.
* 🏏 Virat Kohli is the highest run scorer in IPL history with **8,014 runs**.
* 🎯 Yuzvendra Chahal leads the wicket charts with **205 wickets**.
* 🪙 Winning the toss has minimal influence on the final result, with toss winners winning only **50.59%** of matches.
* 🎯 Captains preferred to **field first in 64.29%** of matches after winning the toss.

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📁 Dataset

The dataset contains IPL match and ball-by-ball data from 2008 to 2024, sourced from Kaggle.
